### 保存数据关机(推荐使用)
> shutdown [选项] 时间
```.bat
-c # 取消前一个关机命令
-h # 关机
-r # 重启

now # 当前时间(格式：12:30)
& # 不占用当前命令行时间
```

> 查看当前时间
```.bat
date
```


### 不保存关机(不推荐)
```.bat
halt
poweroff
init 0
```


### 其他重启命令(不推荐)
```.bat
reboot
init 6
```


### 系统运行级别
* 0.关机
* 1.单用户(类似Windows安全模式、最小启动服务)
* 2.不完全多用户，不含NFS服务
* 3.完全多用户(命令行界面)
* 4.未分配
* 5.图形界面(前提要已经安装)
* 6.重启


### 查询运行级别
```.bat
runlevel
```


### 退出命令
```.bat
logout
```
