## 帮助命令
> manual
```.bat
man [命令]
```
> 如：man man # man ls
```.bat
man -f [命令] # 相当于：whatis [命令]
```
> 查看和命令相关的所有帮助
```.bat
man -k [命令] # 相当于：apropos [命令]
```

> man的级别(使用man后，第一行括号里的数字)
1. 查看命令的帮助
2. 查看可被内核调用的函数的帮助
3. 查看函数和函数库的帮助
4. 查看特殊文件的帮助(主要是/dev目录下的文件)
5. 查看配置文件的帮助
6. 查看游戏的帮助
7. 查看其它杂项的帮助
8. 查看系统管理员可用命令的帮助
9. 查看和内容相关文件的帮助
