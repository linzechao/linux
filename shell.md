### shell
* 命令行解释器
* 功能强大的编程语言
```bash
# linux默认的Shell就是Bash

# 查看系统当前Shell
echo $SHELL
```

#### 输出命令
> echo [e] String
> -e 带符号的语句
```bash
\a # 输出警告音
\b # 退格键
\n # 换行符
\r # 回车符
\t # 制表符
\v # 垂直制表符
\0nnn # 8进制ASCLL码
\xhh # 16进制ASCLL码
```

#### 操作连接符
```bash
;  # 多个操作符
&& # 逻辑操作符
|| # 逻辑操作符
|  # 管道符(前面的命令结果做为后面命令的内容基础)
```
